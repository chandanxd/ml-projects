# Iris Flower Classification

This project implements a basic machine learning classification workflow using the Iris dataset as part of learning supervised learning concepts.

## What was done
- Loaded and explored the Iris dataset
- Split data into training and testing sets
- Trained a classification model using scikit-learn
- Evaluated model performance

## Tools used
- Python
- scikit-learn
- Pandas, NumPy
- Jupyter Notebook

## Notes
This project was completed by following a tutorial and is intended as a learning exercise.
